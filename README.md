## 📖 About

🛒 **PickBazar (Buy-It Clone)** — Modern grocery & beauty e-commerce UI built with **React + Vite + Tailwind CSS**. Features "Groceries Delivered in 90 Minutes" hero, dynamic product grids (fragrances, furniture), category filters, search, **working shopping cart** (qty controls, totals like $5049.97), and **Yup-validated auth** (login/signup).

**Interactive frontend demo** with mock data—showcases React state management, Tailwind utilities, and rapid Vite development. Pure client-side (no real backend/payments).

**Live Demo**: [![Demo](https://img.shields.io/badge/Live%20Demo-buy--it--31bbw.web.app-emerald?logo=vercel)](https://buy-it-31bbw.web.app)

**Key Highlights**:
- 🏠 **Hero Banner**: Fast-delivery promo with integrated search.
- 🗂️ **Categories & Filters**: Beauty, Fragrances, Furniture, Groceries (checkboxes).
- 🛍️ **Products**: Realistic cards (e.g., Calvin Klein $49.99, Gucci Bloom $79.99) with "Add to Cart".
- 🛒 **Cart Sidebar**: Real-time qty (+/-), remove items, running total, "Checkout" CTA.
- 🔐 **Auth Forms**: Yup-powered email/password validation.
- 📱 **Fully Responsive**: Clean, neumorphic design across devices.

**What I Learned**:
- ⚛️ **React + Vite**: Hooks for cart state, fast HMR/builds.
- 🎨 **Tailwind CSS**: Utility-first responsive styling (gradients, cards, hover states).
- ✅ **Yup Validation**: Secure form handling for auth flows.
- 🛒 **State Management**: Global cart updates across components.

**Tech Stack**: React • Vite • Tailwind CSS • Yup • Mock Data (local/JSON) • Vercel
